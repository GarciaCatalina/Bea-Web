# Bea-Web — Sitio Web de beabadoobee

¡Bienvenido/a a **Bea-Web**! Este es un sitio web tributo y de diseño interactivo dedicado a la cantante y compositora filipino-británica **beabadoobee** (Beatrice Laus).

## Características Principales
El sitio está estructurado en base a las secciones clave que representan a la artista, optimizado con un menú de navegación intuitivo y un pie de página completo:

* **Inicio (Index):** Una presentación estética de Bea, sus géneros musicales, su último lanzamiento (*This Is How Tomorrow Moves*) y accesos rápidos.
* **Biografía:** El recorrido detallado de su carrera desde sus inicios en 2017 en Londres, sus influencias musicales (Mazzy Star, Pavement) y su evolución.
* **Discografía:** Línea de tiempo visual de sus trabajos principales, desde sus primeros EPs (*Patched Up*, *Loveworm*) hasta sus álbumes de estudio.
* **Merch (Tienda):** Un catálogo organizado por categorías que brindan acceso directo a la compra oficial.
* **Galería:** Una selección de imágenes que transmiten la identidad visual y las presentaciones en escenario de la artista.
* **Contacto:** Formulario funcional de contacto para fans, prensa o booking.
* La página cuenta con links externos a sus distintas redes oficiales de contacto

---
## Tecnologías Utilizadas
Este proyecto fue desarrollado utilizando tecnologías web fundamentales, priorizando una estructura semántica limpia y un diseño responsivo:

* **HTML5**
* **CSS3**
* **JavaScript**

---

## Estructura del Proyecto

```text
Bea-Web/
├── index.html          # Página principal de presentación
├── biografia.html      # Sección de historia e influencias
├── discografia.html    # Álbumes, EPs y enlaces a plataformas
├── merch.html          # Catálogo de productos y vinilos
├── galeria.html        # Galería de imágenes de la artista
├── contacto.html       # Formulario y canales de comunicación
├── css/                # Archivos de estilos (estructura y diseño)
└── imagenes/           # Recursos visuales, portadas y fotografías
```
